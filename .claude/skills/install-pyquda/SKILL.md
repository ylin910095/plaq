name: install-pyquda
description: How to install PyQUDA
---

When try to install pyquda, follow these instructions:

1. Check the webpage https://github.com/CLQCD/PyQUDA and links within for latest installation instructions
2. Export the appropriate environmental variable so PyQUDA can find QUDA installation and MPI
3. After installation, always run tests to ensure PyQUDA is installed properly. In particular, it can use QUDA
